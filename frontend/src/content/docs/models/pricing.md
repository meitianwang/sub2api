# 模型与定价

## 查看模型与定价

### 方式一：模型页面（无需登录）

访问导航栏中的「模型」页面，可以查看平台所有公开可用的模型信息：

- 模型按分组归类展示
- 显示每百万 Token 的输入价格和输出价格
- 显示每个模型所属的分组
- 可直观对比不同分组中相同模型的定价差异

### 方式二：控制台分组详情（需要登录）

登录后进入控制台，查看分组详情页面：

- 显示每个分组的定价倍率
- 倍率含义说明：
  - **1x** = 与官方定价一致
  - **0.5x** = 官方定价的 50%
  - **0.1x** = 官方定价的 10%
- 倍率越低，实际使用费用越低

### 方式三：通过 API 查询

使用 API Key 查询当前 Key 所属分组的可用模型列表：

```bash
curl {{BASE_URL}}/v1/models \
  -H "Authorization: Bearer sk-xxxx"
```

将 `sk-xxxx` 替换为你的实际 API Key。返回结果为该 Key 所属分组中所有可用模型的列表。

## 计费说明

### 按量计费（标准分组）

标准分组采用按 Token 用量计费的方式，费用从账户余额中实时扣除。

**计费规则：**

- 输入 Token 和输出 Token 分别计价，两者价格可能不同
- 实际费用的计算公式为：

  ```
  实际费用 = 官方模型价格 x 分组倍率
  ```

- 例如，某模型官方输入价格为 ¥3.00/百万 Token，所属分组倍率为 0.5，则实际输入价格为 ¥1.50/百万 Token

**查看使用记录：**

登录控制台，进入「活动日志」页面，可以查看每一次 API 调用的详细信息：

- 调用的模型名称
- 输入/输出 Token 用量
- 本次请求费用
- 请求延迟
- 请求状态（成功/失败）

## 定价示例

假设平台提供以下分组：

| 分组 | 倍率 | 包含模型 |
|------|------|---------|
| 分组 A | 0.5x | claude-sonnet-4-6, claude-opus-4-6 |
| 分组 B | 0.1x | claude-sonnet-4-6, gpt-5.4 |

以 claude-sonnet-4-6 为例，假设官方价格为输入 ¥3.00/百万 Token、输出 ¥15.00/百万 Token：

- 在分组 A（倍率 0.5x）中的实际价格：
  - 输入：¥3.00 x 0.5 = ¥1.50/百万 Token
  - 输出：¥15.00 x 0.5 = ¥7.50/百万 Token

## 余额管理

- 充值后余额立即到账，可在控制台首页查看当前余额
- 每次 API 调用完成后，费用实时从余额中扣除
- 余额不足时，API 请求将返回错误，请及时充值
- 可在创建 API Key 时设置额度上限，防止单个 Key 消耗过多余额
