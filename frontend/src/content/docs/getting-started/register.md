# 注册账号

本平台支持通过邮箱注册账号。完成注册后即可使用平台提供的 API 服务。

## 注册步骤

1. 访问注册页面：`{{BASE_URL}}/register`
2. 填写注册信息：
   - **邮箱地址** -- 请使用常用邮箱，后续用于登录和接收通知
   - **密码** -- 建议使用 8 位以上、包含字母和数字的强密码
   - **邀请码 / 优惠码**（如有）-- 部分站点可能要求填写邀请码，或支持通过优惠码获得额外额度，具体取决于管理员配置
3. 点击「注册」按钮提交
4. 如平台开启了邮箱验证，请前往邮箱查收验证邮件，点击链接完成验证
5. 验证通过后，使用邮箱和密码登录即可进入控制台

## 注意事项

- 请妥善保管账号密码，不要将密码分享给他人
- 如果未收到验证邮件，请检查垃圾邮件文件夹
- 忘记密码时可通过登录页面的「忘记密码」功能重置
