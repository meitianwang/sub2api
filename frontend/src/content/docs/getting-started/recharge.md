# 充值余额

平台采用预充值模式，API 调用时会根据实际 Token 用量从账户余额中扣费。请确保账户中有足够余额后再发起调用。

## 充值方式

### 在线充值

1. 登录后进入控制台
2. 点击侧边栏的「钱包」或「充值」入口
3. 选择充值金额和支付方式（具体支付方式由管理员配置，可能包括支付宝、微信等）
4. 完成支付后，余额会即时到账

### 兑换码充值

1. 如果持有兑换码（由管理员或活动发放），进入控制台的「兑换码」页面
2. 输入兑换码并提交
3. 对应额度将立即充入账户余额

## 计费说明

- 计费基于 **Token 用量**，不同模型的单价不同
- 输入 Token 和输出 Token 分别计价，输出 Token 通常单价更高
- 余额一经充值，**不会过期**
- 建议前往「模型与定价」页面查看各模型的详细费率，合理规划用量

## 余额查看

登录控制台后，可在首页或钱包页面随时查看当前余额和消费明细。
